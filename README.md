# innowise-lab-internship

# Getting started

- Clone the repository

```
git clone  https://github.com/JekEugene/innowise-lab-internship <project_name>
```

- Install dependencies

```
cd <project_name>
npm install
```

- Start your PostgreSQL server

- Run the project

```
npm run start
```
